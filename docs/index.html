1부터 9까지 서로 다른 수로 이루어진 3자리의 수를 맞추는 게임이다.
숫자는 맞지만 위치가 틀렸을 때는 볼.
숫자와 위치가 전부 맞으면 스트라이크.
ex) 정답이 425일 때,
123을 제시한 경우 : 1 스트라이크
456을 제시한 경우 : 1 볼 1 스트라이크
789를 제시한 경우 : 0 볼 0 스트라이크
이 같은 과정을 반복해 3개의 숫자를 모두 맞히면 게임이 종료된다.

1 2 3 4 5 6 7 8 9 R B S
COM 0 0 0 0
YOU 0 0 0

입력값은
0 볼 0 스트라이크


B ● ● ●
S ● ● ●
